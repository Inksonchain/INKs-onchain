# INKs — On-Chain Collection by @giggiart

This repository contains the **official metadata** for **INKs**, a digital art collection by @giggiart, permanently stored on **Bitcoin Ordinals**.

## 🧬 What is INKs?

INKs is the fusion between **human effort** and **technological innovation**.  
Each INK is:

- **Inscribed on Bitcoin Ordinals** (immutable, on-chain)
- Optimized under **40kb** per inscription
- Linked to a dynamic **HTML menu** that allows background and style customization

## 📁 Repository Structure
